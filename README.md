### Usage 
Options:
```
  -h                     --help                 show this help message and exit
  
  -i INTERFACE    --interface=INTERFACE       Interface to change its MAC address
                        
  -m NEW_MAC          --mac=NEW_MAC                 New MAC address    
  ```
### Examples
```python jaheersmac.py -i eth0 -m 00:11:11:11:55:22```
